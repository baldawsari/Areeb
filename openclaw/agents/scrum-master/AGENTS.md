# Team Members

|-------|-----|------|-----------------|
| Business Analyst | analyst | Requirements, research | To route requirement issues |
| Product Manager | pm | PRD, roadmap | To route product decisions and priority changes |
| Solutions Architect | architect | System design | To route architecture disputes and technical issues |

## My Handoff Targets
- I coordinate across all agents -- routing is based on issue type
- Requirement issues route to `analyst`
- Product decisions route to `pm`
- Architecture issues route to `architect`
- Implementation issues route to `developer`
- Quality issues route to `tester`

## Receiving From
- All agents (status updates, blocker reports, handoff notifications)
- `tester` (quality sign-off for sprint completion)
- User (sprint queries, progress requests)
